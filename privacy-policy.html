1. 개요

배달의비번(이하 "앱")은 배달 라이더를 위한 주소 및 비밀번호 관리 서비스를 제공합니다. 본 개인정보 처리방침은 앱 사용 시 수집, 이용, 보관, 공유되는 개인정보에 대해 설명합니다.

본 앱은 사용자의 개인정보 보호를 최우선으로 생각하며, 개인정보보호법 및 정보통신망 이용촉진 및 정보보호 등에 관한 법률을 준수합니다.

2. 정보 수집 방법 및 수집하는 정보

2.1 개인용 모드 (오프라인 모드)

개인용 모드를 사용하는 경우, 앱은 어떠한 개인정보도 수집하거나 외부로 전송하지 않습니다.

  • 모든 데이터는 사용자의 기기(로컬 SQLite 데이터베이스)에만 저장됩니다
  • 계정 생성이 필요하지 않습니다
  • 인터넷 연결 없이 완전히 오프라인으로 작동합니다
  • 개발자는 사용자의 데이터에 접근할 수 없습니다

2.2 그룹용 모드 (클라우드 동기화 모드)

그룹용 모드를 사용하는 경우, 팀 협업 기능을 위해 다음과 같은 정보를 수집합니다:

필수 수집 정보

계정 정보 (회원가입 시):

  • 이메일 주소
  • 비밀번호 (암호화 저장)
  • 가입 일시

주소 정보 (사용자가 직접 입력):

  • 위도 및 경도 좌표
  • 도로명 주소 또는 지번 주소
  • 건물명 (선택)
  • 사용자가 작성한 메모 (선택)

비밀번호 정보 (사용자가 직접 입력):

  • 아파트 공동현관 비밀번호
  • 메모 (선택)

팀 정보:

  • 팀 이름
  • 초대 코드
  • 팀 멤버십 정보

선택 수집 정보

GPS 추적 데이터 (GPS 추적 기능 사용 시):

  • 실시간 위치 좌표 (위도, 경도)
  • 이동 경로
  • 이동 거리 및 시간
  • 속도 정보

사용 통계:

  • 방문 횟수
  • 마지막 방문 시간
  • 즐겨찾기 설정

자동 수집 정보

기기 정보:

  • 기기 모델명
  • 운영체제 버전 (Android)
  • 앱 버전
  • 언어 설정

로그 데이터:

  • 앱 사용 오류 로그
  • 앱 충돌 보고서 (Google Play Console을 통해 자동 수집)

3. 정보 수집 목적

수집한 정보는 다음과 같은 목적으로만 사용됩니다:

3.1 필수 서비스 제공

  • 계정 관리: 회원 가입, 로그인, 본인 확인
  • 주소 관리: 배달 주소 및 비밀번호 저장, 조회, 수정, 삭제
  • 위치 기반 알림: 저장된 주소 근처 도착 시 자동 알림 제공
  • 팀 협업: 팀 생성, 멤버 관리, 데이터 실시간 동기화

3.2 서비스 개선

  • 사용 통계 분석: 자주 방문하는 주소 표시, 개인화된 사용 경험 제공
  • GPS 추적: 배달 경로 기록 및 통계 제공 (사용자 선택 기능)
  • 버그 수정: 앱 오류 및 충돌 원인 파악 및 해결

3.3 고객 지원

  • 문의 응대: 사용자 문의 및 불만 처리
  • 계정 복구: 비밀번호 재설정 지원

4. 정보 보관 및 저장

4.1 개인용 모드

  • 모든 데이터는 사용자 기기의 로컬 SQLite 데이터베이스에 저장됩니다
  • 앱 삭제 시 모든 데이터가 함께 삭제됩니다
  • 개발자는 사용자 데이터에 접근할 수 없습니다

4.2 그룹용 모드

  • 계정 정보 및 공유 데이터는 Supabase 클라우드 서버에 저장됩니다
  • Supabase는 AWS 인프라를 사용하며, 데이터는 암호화되어 저장됩니다
  • 서버 위치: AWS 아시아 태평양 (서울) 리전

4.3 보관 기간

  • 계정 정보: 회원 탈퇴 시까지
  • 주소 및 비밀번호 정보: 사용자가 삭제하거나 계정 탈퇴 시까지
  • GPS 추적 데이터: 사용자가 삭제하거나 계정 탈퇴 시까지
  • 로그 데이터: 수집 후 최대 90일

계정 탈퇴 시 모든 개인정보는 즉시 삭제되며, 복구할 수 없습니다.

5. 정보 공유 및 제3자 제공

5.1 정보 공유 원칙

배달의비번은 사용자의 개인정보를 제3자에게 판매하거나 공유하지 않습니다.

5.2 제3자 서비스 제공업체

다음의 제3자 서비스를 사용하며, 각 서비스는 자체 개인정보 처리방침을 가지고 있습니다:

Supabase (클라우드 데이터베이스 및 인증)

  • 목적: 그룹용 모드 계정 관리, 데이터 동기화
  • 전송 정보: 이메일, 암호화된 비밀번호, 주소 데이터, 팀 정보
  • 개인정보 처리방침: https://supabase.com/privacy

Google Sign-In (OAuth 로그인)

  • 목적: 간편 로그인 제공
  • 전송 정보: Google 계정 이메일, 프로필 정보
  • 개인정보 처리방침: https://policies.google.com/privacy

Naver Map API (지도 서비스)

  • 목적: 지도 표시, 주소 검색, 역지오코딩
  • 전송 정보: 위도/경도 좌표, 주소 검색어
  • 개인정보 처리방침: https://www.navercorp.com/policy/privacy

Google Play Services (위치 서비스)

  • 목적: GPS 위치 추적, 근처 주소 감지
  • 전송 정보: 실시간 위치 좌표
  • 개인정보 처리방침: https://policies.google.com/privacy

5.3 법적 요구 사항

다음의 경우 법적 의무를 준수하기 위해 정보를 공개할 수 있습니다:

  • 법원의 명령 또는 법적 절차에 따른 요구
  • 법률에 의해 요구되는 경우
  • 사용자 또는 타인의 권리, 재산, 안전을 보호하기 위해 필요한 경우

6. 정보 보안

6.1 기술적 보안 조치

  • 암호화 전송: HTTPS/TLS 프로토콜을 통한 데이터 전송
  • 비밀번호 암호화: bcrypt 알고리즘을 사용한 비밀번호 해싱
  • 데이터베이스 암호화: Supabase Row Level Security (RLS) 정책 적용
  • 접근 제어: 팀 멤버만 해당 팀의 데이터에 접근 가능

6.2 관리적 보안 조치

  • 개인정보 취급자 최소화
  • 정기적인 보안 점검 및 업데이트
  • 개인정보 유출 사고 대응 절차 마련

6.3 사용자 책임

사용자는 다음 사항에 유의하여야 합니다:

  • 타인과 계정(이메일/비밀번호)을 공유하지 마세요
  • 안전한 비밀번호를 설정하세요 (8자 이상, 영문+숫자+특수문자 조합)
  • 기기 분실 시 즉시 비밀번호를 변경하세요
  • 공용 기기에서 로그아웃을 잊지 마세요

7. 사용자 권리

사용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

7.1 개인정보 열람 및 수정

  • 앱 설정 메뉴에서 계정 정보 확인 및 수정 가능
  • 주소 및 비밀번호는 언제든지 추가, 수정, 삭제 가능

7.2 개인정보 삭제

  • 개별 데이터 삭제: 앱 내에서 주소, 비밀번호, GPS 추적 기록 개별 삭제
  • 전체 데이터 삭제: 설정 > 데이터 관리 > 모든 데이터 초기화

7.3 계정 탈퇴

  • 설정 > 계정 > 회원 탈퇴
  • 탈퇴 시 모든 개인정보 즉시 삭제 (복구 불가)
  • 팀 오너인 경우, 팀을 먼저 삭제하거나 다른 멤버에게 양도해야 합니다

7.4 정보 이동권 (데이터 다운로드)

  • 설정 > 데이터 관리 > 내보내기
  • JSON 형식으로 모든 데이터 다운로드 가능
  • 다른 기기로 데이터 이동 가능

7.5 동의 철회

  • 위치 권한: Android 설정 > 앱 > 배달의비번 > 권한에서 위치 권한 해제
  • 알림 권한: Android 설정 > 앱 > 배달의비번 > 알림에서 알림 해제
  • 그룹용 모드 해제: 앱 설정에서 개인용 모드로 전환 (계정 유지)

8. 위치 정보 수집 및 사용

8.1 위치 권한

앱은 다음과 같은 위치 권한을 요청합니다:

  • 정확한 위치 (GPS): 주소 등록, 근처 주소 감지, GPS 추적에 사용
  • 백그라운드 위치 (선택): 앱이 백그라운드에 있을 때 근접 알림 제공

8.2 위치 정보 사용 목적

  • 주소 등록: 현재 위치를 기반으로 주소 자동 입력
  • 근처 주소 감지: 저장된 주소 100m 이내 접근 시 알림 제공
  • GPS 추적: 배달 경로 기록 (사용자가 명시적으로 활성화한 경우만)

8.3 위치 정보 보관

  • 주소 정보 (위도/경도)는 사용자가 삭제하거나 계정 탈퇴 시까지 보관
  • GPS 추적 데이터는 사용자가 삭제할 때까지 보관
  • 실시간 위치는 저장되지 않으며, 근접 감지 목적으로만 임시 사용됨

8.4 위치 권한 거부

  • 위치 권한을 거부해도 앱의 기본 기능(주소 및 비밀번호 관리)은 사용 가능합니다
  • 단, 자동 알림 및 GPS 추적 기능은 사용할 수 없습니다

9. 아동의 개인정보 보호

본 앱은 만 18세 미만의 아동을 대상으로 하지 않습니다.

  • 배달 업무 종사자를 대상으로 설계되었습니다
  • 고의로 만 18세 미만 아동의 개인정보를 수집하지 않습니다
  • 아동의 개인정보가 수집된 것으로 확인될 경우 즉시 삭제합니다

10. 쿠키 및 유사 기술

본 앱은 웹 기반이 아니므로 쿠키를 사용하지 않습니다.

다만, 다음과 같은 로컬 저장소를 사용합니다:

  • SQLite 데이터베이스: 주소, 비밀번호, GPS 추적 데이터 저장
  • SharedPreferences: 앱 설정(모드, 마지막 동기화 시간 등) 저장

이러한 로컬 저장소는 사용자의 기기에만 존재하며, 외부로 전송되지 않습니다.

11. 국제 데이터 전송

11.1 데이터 저장 위치

  • 개인용 모드: 사용자 기기 내부 (전송 없음)
  • 그룹용 모드: Supabase 서버 (AWS 서울 리전)

11.2 국외 전송

그룹용 모드 사용 시, 일부 데이터가 다음 국가로 전송될 수 있습니다:

  • 미국: Supabase 본사 서버 (백업 및 관리 목적)
  • EU: Google OAuth 인증 서버

모든 국외 전송은 HTTPS/TLS 암호화를 통해 안전하게 이루어집니다.

12. 개인정보 처리방침 변경

본 개인정보 처리방침은 법률 또는 서비스 변경에 따라 수정될 수 있습니다.

12.1 변경 통지

  • 변경 사항이 있을 경우 앱 내 공지 및 이메일로 통지합니다
  • 중요한 변경(수집 정보 추가, 목적 변경 등)의 경우 최소 7일 전 사전 통지합니다
  • 변경된 개인정보 처리방침은 시행일부터 효력이 발생합니다

12.2 동의 철회

  • 변경 사항에 동의하지 않을 경우 서비스 이용을 중단하고 계정을 탈퇴할 수 있습니다
