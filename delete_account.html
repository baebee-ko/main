⚠️ 중요 안내

계정을 삭제하면 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다. 신중하게 결정해 주시기 바랍니다.

1. 앱 내에서 계정 삭제

배달의비번 앱에서 직접 계정을 삭제할 수 있습니다.

1
앱 실행
배달의비번 앱을 실행합니다.
2
설정 메뉴 열기
하단 내비게이션 바에서 '설정' 탭을 선택합니다.
3
계정 관리
'계정 관리' 메뉴로 이동합니다.
4
계정 삭제 버튼
'계정 삭제' 버튼을 클릭합니다.
5
확인
삭제 확인 대화상자에서 '삭제' 버튼을 클릭하여 최종 확인합니다.

💡 개인용 모드 사용자

개인용 모드(오프라인 모드)를 사용하는 경우 계정이 없으므로 앱을 삭제하면 모든 데이터가 제거됩니다. 별도의 계정 삭제 절차가 필요하지 않습니다.

2. 이메일로 계정 삭제 요청

3. 삭제되는 데이터

계정 삭제 시 다음 데이터가 영구적으로 삭제됩니다:

데이터 유형 삭제 여부 설명
계정 정보 즉시 삭제 이메일, 비밀번호, 프로필 정보
주소 정보 즉시 삭제 저장된 모든 배달 주소
비밀번호 정보 즉시 삭제 저장된 건물 출입 비밀번호
팀 데이터 즉시 삭제 팀 멤버십 정보 (팀 자체는 유지)
GPS 추적 기록 즉시 삭제 배달 경로 및 위치 기록
알림 로그 즉시 삭제 위치 기반 알림 이력
백업 데이터 30일 후 삭제 서버 백업본 (복구 불가능하게 처리)

4. 보관되는 데이터

법적 의무 또는 정당한 사유로 다음 데이터는 일정 기간 보관됩니다:

데이터 유형 보관 기간 보관 사유
구매 이력 5년 전자상거래법 (세금 신고 목적)
결제 기록 5년 국세기본법 (거래 기록 보관 의무)
서비스 이용 기록 3개월 통신비밀보호법 (로그 기록 보관)
부정 이용 기록 1년 부정 이용 방지 및 재가입 제한

📋 법적 근거

  • 전자상거래 등에서의 소비자보호에 관한 법률 - 계약 또는 청약철회 등에 관한 기록: 5년
  • 국세기본법 - 세법이 규정하는 모든 거래에 관한 장부 및 증빙서류: 5년
  • 통신비밀보호법 - 컴퓨터통신, 인터넷 로그기록자료, 접속지 추적자료: 3개월

5. 개인용 모드 데이터 삭제

개인용 모드(오프라인 모드)를 사용하는 경우:

1
앱 삭제
기기에서 배달의비번 앱을 삭제하면 모든 로컬 데이터가 함께 삭제됩니다.
2
또는 앱 내 데이터 초기화
설정 → 데이터 관리 → 전체 데이터 초기화를 선택합니다.

⚠️ 주의사항

  • 개인용 모드의 데이터는 기기에만 저장되므로 서버에 계정 정보가 없습니다
  • 앱 삭제 후에는 데이터를 복구할 수 없습니다
  • 백업 기능을 사용하여 미리 데이터를 내보내기 하는 것을 권장합니다

6. 계정 삭제 후 재가입

계정 삭제 후 언제든지 새로운 계정으로 다시 가입할 수 있습니다. 단, 이전 데이터는 복구되지 않으며 새로운 계정으로 시작하게 됩니다.

💡 재가입 제한

서비스 부정 이용 이력이 있는 경우, 계정 삭제 후 1년간 재가입이 제한될 수 있습니다.
